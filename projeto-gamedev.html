Game corrida pela portalgun

Inicio: 06/2020
Fim: Ainda em desenvolvimento

Sobre o projeto

Esse projeto foi uma proposta da Alura cursos online, em sua semana de imerssão em desenvolvimento de games, nessa semana os participantes foram introduzidos aos conceitos no desenvolvimentos de games e então fomos instruídos na lógica principal do modelo de jogo endless runner, que consiste em um personagem que segue o jogo desviando de obstáculos.

Nesse projeto foi utilizada a plataforma P5.js para realizar configurações de infraestrutura e poupar o desenvolvimento dessa parte para focar no game.

Por fim, uma parte muito importante dos games é o history telling, então confiram o que eu criei e usei para desenvolver tudo: "O jogo é baseado na série Rick and Morty no qual o personagem principal (eu mesmo kk) tento pegar a portalgun "emprestada" mas como usar ela é bem dificil acabo indo para um mundo desconhecido... e o Rick veio logo atrás. O objetivo é fugir dos raios laser que o Rick está disparando até que o personagem consiga retornar ao seu mundo."

Técnologias

P5.js, Javascript, Css e Html

Desafios

O meus maiores desafios nesse projeto foi entender como realizar as colisões entre os personagens do game, entender a mecânica de aplicar a gravidade sobre os itens e também fazer o desenho de todas as sprites do jogo.

Status

A semana de imersão instruiu até certo ponto, mas eu tenho planos para implementar ideias que eu tive durante o desenvolvimento, uma delas por exemplo, seria usar a temática da portalgun para alterar fases, trocando de mundos e etc.
