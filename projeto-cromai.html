Sobre o projeto

Esse projeto foi proposto para participação de um processo seletivo, onde o objetivo era resolver o case (para mais detalhes conferir o repositório no GitHub, pois la tem o case) que foi enviado e basicamente pedia para desenvolver uma calculadora que consumia uma API externa.

Técnologias

Javascript, AJAX, Css e Html

Desafios

O maior desafio desse projeto foi entender como as requisições funcionam e como elas devem ser feitas, pois foi meu primeiro contato com essa tecnologia.

Status

O projeto está finalizado (tudo está disponibilizado no meu GitHub) e foi enviado para a empresa. Para futuros testes tenha em mente que a API era da empresa que propôs o case, então pode estar fora do ar.
