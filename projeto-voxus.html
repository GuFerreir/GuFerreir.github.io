Projeto de BackEnd para Voxus

Inicio: 06/2020
Fim: 06/2020

Sobre o projeto

Esse projeto foi proposto para participação de um processo seletivo, onde o objetivo era desenvolver uma API e consumi-la. Então foi necessário realizar tanto o Front End quanto o Back End para a entrega.

A aplicação deveria ser capaz de cadastrar dados de um pagamento e fazer a persistencia desses dados, alem disso ela tambem deveria ser capaz de fazer upload de arquivos em .xls (devidamente formatado) e inserir os dados desse arquivo em sua base de dados e exibí-los na interface.

Técnologias

Javascript, NodeJS, MySQL, Bootstrap, HTML e Conceitos de REST

Desafios

Foi a primeira API que eu desenvolvi, então o maior desafio foi aprender e utilizar os padrões REST na arquitetura do Back End.

Status

O projeto já está finalizado, bem documentado (tudo está disponibilizado no meu GitHub) e já foi entregue para a empresa.
